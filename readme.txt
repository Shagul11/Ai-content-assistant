AI Content Assistant is a professional WordPress plugin developed by Sayed Shagul that helps you generate high-quality content for your website . Create blog posts, product descriptions, social media captions, and SEO meta content directly from your WordPress admin.

== Description ==

**AI Content Assistant** allows WordPress users to quickly generate content using artificial intelligence. It is ideal for bloggers, marketers, and e-commerce website owners who want professional content fast.

**Key Features:**

* Generate blog posts with headings, subheadings, short paragraphs, and call-to-actions (CTAs)
* Create SEO-friendly product descriptions with features, benefits, and use-cases
* Write social media captions with hashtags and short CTAs
* Generate SEO meta titles, meta descriptions, and excerpts
* Customize tone, length, and add extra instructions for AI output
* Live preview of formatted AI content before insertion
* Copy, download, or insert content directly into Gutenberg editor
* Works with any valid OpenAI API key

**Why Use AI Content Assistant?**

This plugin saves time and effort by generating professional, clear, and structured content. It ensures consistency in tone and style, making your site content engaging and reader-friendly.

== Installation ==

1. Upload the `ai-content-assistant` folder to the `/wp-content/plugins/` directory, or use the WordPress admin plugin upload feature.
2. Activate the plugin via the **Plugins** menu in WordPress.
3. Navigate to **AI Content Assistant** in the WordPress admin menu.
4. Enter your OpenAI API key and start generating content immediately.

== Frequently Asked Questions ==

= Do I need an OpenAI account? =
Yes, you need a valid OpenAI API key to use this plugin.

= Can I customize the tone and length of the generated content? =
Yes, you can select tone, length, and add extra instructions for each generation.

= Can I insert generated content directly into posts? =
Yes, the plugin supports direct insertion into the Gutenberg editor.

= Is my API key secure? =
Your API key is stored securely in WordPress options. You can also define it in `wp-config.php` for additional security.
